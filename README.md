## LCCSqliteManager
这是一个Sqlite数据库框架
github地址:https://github.com/OldChicken/LCCSqliteManager

## 目的
这是一个菜鸟新手第一个封装的框架！！这是一个菜鸟新手第一个封装的框架！！这是一个菜鸟新手第一个封装的框架！！

## 特点
这是一个轻量级框架，目的是让使用的人脱离Sql语句。你要建表，传入几个参数就可以，删除一条数据，传入表名和一个删除条件数组
就能达到目的，我把大部分Sql语句帮你封装好了。
